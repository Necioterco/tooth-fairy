Moonlit

Schedule Claude desktop runs against your local repo.

A tiny menu bar app that drives the Claude desktop app on your Mac at a scheduled time. For tasks that need access to your actual files — your branch with uncommitted changes, your local MCP servers, your real dev environment.

Download for macOS View on GitHub

Free · macOS 13+ · Requires Claude desktop

💡
Already have Routines? If your task doesn't need local files, use those instead — they run in Anthropic's cloud, on a real cron, even when your Mac is off. Moonlit is for the cases Routines can't reach: your local code, your machine, your environment.

Real local folders

Points at a folder on your Mac. Claude opens it as a project on your actual machine — uncommitted changes, local MCP servers, the works.

⚡

Auto mode

Optional toggle flips Claude into Auto mode before sending so long unattended runs don't stop on every confirm prompt.

⏰

Scheduled or instant

Schedule a one-shot for 10pm, or hit Run now. Pending, running, succeeded, failed — all tracked in a clean log.

🪶

Lives in the menu bar

No dock icon. No window juggling. Click the moon, write a prompt, go to bed.

Rule of thumb: if the task could run on a fresh cloud sandbox with just a GitHub clone, use Routines. If it needs your branch, your services, your tools — use Moonlit.

How it works

  1. Pick a folder — the project Claude should open.
  2. Set the time — when Moonlit should fire the prompt.
  3. Write the prompt — exactly what you'd type into Claude.
  4. Optionally enable Auto mode — for long unattended runs.
  5. Sleep — Moonlit handles the rest.
