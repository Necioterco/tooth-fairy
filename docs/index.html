One-shot, your time

Pick any moment — tonight at 10pm, Saturday at noon. No cron, no recurring, no fuss. Trigger early with Run now whenever.

📂

Real local files

Points at a folder on your Mac. Claude opens it as a project before each run — your branch, your services, your tools.

💬

Sessions, not silos

The result lands in your normal Claude history. Continue it from your phone or web — same place every other chat lives.

⚡

Your subscription

Runs through Claude desktop on your Pro or Max plan. No API tokens billed, no extra costs. Optional Auto mode for long unattended runs.

How it works

  1. Pick a folder — the project Claude should open.
  2. Set the time — when the prompt should fire.
  3. Write the prompt — exactly what you'd type into Claude.
  4. Sleep — Tooth Fairy handles the rest.
