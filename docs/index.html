How it works

  1. Pick a folder — the project Claude should open.
  2. Set the time — when the prompt should fire.
  3. Write the prompt — exactly what you'd type into Claude.
  4. Sleep — Tooth Fairy handles the rest.

What it is — and what it isn't

  1. It's a local automation. Tooth Fairy types into Claude desktop the same way you would, using the macOS Accessibility APIs you grant on first launch.
  2. It uses your account. Sessions run on your Pro or Max plan, in your normal Claude history. No tokens are billed, nothing is proxied through a server.
  3. It isn't a jailbreak or a bot farm. One scheduled prompt at a time, on your own machine. Don't use it to circumvent rate limits or run mass parallel agents — that's against Anthropic's usage policy.
  4. Auto mode is your call. If you enable it for unattended runs, Claude can execute tools without confirmation. Point it at projects where that's safe (clean branch, no production credentials) and treat it like any other automation.
  5. Open source. Read the code, build it yourself, or fork it — github.com/Necioterco/tooth-fairy.
  6. Not affiliated with Anthropic. "Claude" is Anthropic's trademark; Tooth Fairy is an independent third-party tool that works alongside their desktop app.
