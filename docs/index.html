Moonlit

Schedule overnight Claude desktop runs.

A tiny Mac menu bar app that picks the project folder, switches to Auto mode, pastes your prompt, and hits send — while you sleep.

Download for macOS View on GitHub

Free · macOS 13+ · Requires Claude desktop

📂

Folder-based projects

Point Moonlit at a folder. It opens that folder in Claude before each run — no fragile name matching.

⚡

Auto mode

Optional toggle flips Claude into Auto mode before sending. Long-running tasks run uninterrupted.

⏰

Scheduled or instant

Schedule for 10pm or run now. Pending, running, succeeded, failed — all tracked in a clean log.

🪶

Lives in the menu bar

No dock icon. No window juggling. Click the moon, write a prompt, go to bed.

How it works

  1. Pick a folder — the project Claude should open.
  2. Set the time — when Moonlit should fire the prompt.
  3. Write the prompt — exactly what you'd type into Claude.
  4. Optionally enable Auto mode — for long unattended runs.
  5. Sleep — Moonlit handles the rest.
